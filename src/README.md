# src フォルダ（開発用の原本バックアップ）

このフォルダは、アプリの「編集の大元」と「問題データの原本」を保管する場所です。
公開されているアプリ（GitHub Pages）は、リポジトリ直下の `index.html` です。
**このフォルダのファイルは公開アプリには影響しません**（バックアップと編集の元データ）。

## ファイルの役割

| ファイル | 役割 |
|---|---|
| `仕訳学習アプリ.html` | **開発マスター版（v3.13）**。問題データは内蔵していない。アプリの編集は必ずこのファイルに対して行う。 |
| `問題セット_第1問特集①-⑨.json` | **問題データの原本（49問）**。アプリの「設定 → 問題セットを読み込む」で取り込める形式。 |

## なぜこのフォルダがあるのか

以前はマスター版HTMLと問題JSONが依頼者さんのPC1台にしか無く、PCが壊れると
復旧できないリスクがありました。GitHubに置いておくことで、いつでも取り戻せます。

※このフォルダの2ファイルは、公開中の `index.html`（当時の配布版）から
　機械的に復元したものです（配布版から埋め込みデータを取り出し・版数表記を戻したもの）。

## 配布版（公開アプリ）の更新手順（おさらい）

1. `src/仕訳学習アプリ.html` を編集し、ヘッダーの版数表記（v3.x）を上げる。
2. マスター版の `const BUNDLED_SETS=[];` を、問題JSONを注入した
   `const BUNDLED_SETS=[<問題セットJSON>];` に置き換える。
3. 版数表記を「vX.X配布版」にして、リポジトリ直下の `index.html` として保存（上書き）。
4. GitHub にアップロードすると、数分で全学生に反映される。
